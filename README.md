# FeedBack
Feedback system for avc college
# to run this project 
first install requriments packages
<br> 
`pip install -r requirements.txt`
<br>
then run django server 
<br>
`python manage.py runserver`
<br>
and go to our broswer
<br>
`http://127.0.0.1:8000/`
<br>
paste this url 
<br>
